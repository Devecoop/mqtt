# Lelylan MQTT Server

MQTT server for Lelylan


## Requirements

The MQTT node is tested against Node 0.8.8.


## Getting Started

* Clone `git clone git@github.com:lelylan/mqtt.git`
* Run `npm install && npm install -g foreman`
* Run `mongod` (MongoDB up and running)
* Run `nf start`
